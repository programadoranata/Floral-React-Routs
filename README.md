# Floral-React-Routs

Projeto temático desenvolvido em React com Vite para explorar a navegação em **Single Page Applications (SPA)** utilizando o **react-router-dom**. A interface é inspirada em um tema floral, com cores suaves e um visual leve, proporcionando uma experiência agradável e organizada.

## Sobre o Projeto

O objetivo do projeto é praticar a configuração de rotas em uma aplicação React moderna. O usuário pode navegar entre as páginas **Home**, **Sobre** e **Contato** de forma dinâmica, sem que a aplicação seja recarregada, garantindo transições fluidas.

## Funcionalidades

* Navegação SPA utilizando React Router
* Troca de páginas sem recarregamento do navegador
* Estrutura simples com componentes separados por páginas
* Tema floral com visual clean e delicado

Projeto desenvolvido para fins educacionais.
